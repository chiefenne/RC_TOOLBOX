# Getting started

## Prerequisites

- An ESP32 board (ESP32 DevkitC-1 recommended)
- TFT touch display (2.4" TFT touch display with ILI9341 driver))
- USB cable and a flashing/upload workflow you already use (e.g., mpremote, rshell, Thonny)

## First run checklist

1. Flash the firmware to the board.
2. Copy the project files to the device.
3. Reboot; the UI should start from `main.py`.
