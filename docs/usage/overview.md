# Overview

RC TOOLBOX provides a touch-based graphical interface with rotary encoder support for precise control.

## Navigation

### Touch

- **Tap** buttons and menu items to select
- **Tap** servo selection buttons (1-6) to toggle individual servos
- **Swipe** sliders to adjust values

### Rotary Encoder

The EC11 rotary encoder provides three inputs:

| Action | Function |
|--------|----------|
| **Rotate** | Navigate between UI elements / adjust values |
| **Press (click)** | Select / confirm |
| **Double-click** | Go back / cancel |
| **Triple-click** | Return to home screen |
| **Long press** | Context action (e.g., select all servos) |

### Focus Navigation

When using the encoder, a highlighted outline shows the currently focused element. Rotate to move focus between buttons, sliders, and controls. Press to activate the focused element.

## Home Screen

![Home Screen](../assets/Home_screen.png)

The home screen shows all available tools:

- **Servo** – Servo tester (up to 6 channels)
- **Lipo** – Battery voltage checker
- **CG** – Center of gravity scale
- **Deflection** – Control surface deflection measurement
- **Angle** – Angle of incidence measurement
- **About** – Version and credits

Tap a button or use the encoder to select a tool.
